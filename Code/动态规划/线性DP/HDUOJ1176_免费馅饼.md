# 💡
可以看作是一个倒着的数塔，每个点连接三条边  
其中高度就是它给的时间，毕竟时间越长离地面越远  
然后我们记录一下出现过的最高高度，就当作塔底遍历起点  
每一个单位时间可以移动左一格、右一格、不动。  
所以得到三条边的递推式为：dp[i][j] += MAX(dp[i + 1][j], MAX(dp[i + 1][j + 1], dp[i + 1][j - 1]));
