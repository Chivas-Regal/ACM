# 🔗
**题目描述**  
给定 n 个整数（数字可能重复），求在这些数中选取任意个，使得他们的异或和最大。  
  
**输入格式**  
第一行一个数 n，表示元素个数  
  
接下来一行 n 个数  
  
**输出格式**  
仅一行，表示答案。  
  
**输入输出样例**  
**输入**  
2  
1 1  
**输出**  
1  
**说明/提示**
