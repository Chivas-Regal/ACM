# 💡
我们顺序枚举最小值  
换句话讲就是每次让a[i]作为一个区间的最小值  
  
用单调栈得出每个a[i]区间的左右边界   
然后维护 a[i] * 区间和 的最大值（区间和？前面加上前缀和
