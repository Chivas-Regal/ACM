# 💡
我们都想让走的步数最少所以用最大距离，首先有两种情况：  
1. x 小于最大距离，那我们如果可以找到一个存在的与 x 相等的距离，那么就1，否则就2  
2. x 大于等于最大距离，那我们可以通过折步来使 d 次移动的贪心距离控制在 ((d - 1) * max, d * max]，所以利用 x / max 向上取整(也就是 (x + max + 1) / max) 计算此时的步数
