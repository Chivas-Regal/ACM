# 💡
题目让找满足 a[i] + b[j] + c[k] = x    
所以我们改一下式子变成：a[i] + b[j] = x - c[k]  
  
即我们对 a[i] + b[j] 打一个ab表  
然后排序后每输入一个x  
就枚举c[i]然后在ab表内二分查找元素是否存在
